INSTALL SCRIPT: Standalone x32 or x64:
--------------------------------------
You can copy the Monique 32x or 64x folder to any location you like. 
To create a desktop shortcut after copying you can use the script 
CREATE-DESKTOP-ICON-FOR-STANDALONE.

INSTALL SCRIPT: VST x32 or x64:
--------------------------------------
You can copy the VST for your OS arch or whatever your VST host supports
manually or you can use the script COPY-VST-TO-SYSTEM-VST-FOLDER (just 
use the right one for your OS arch (x32 or x64)).
